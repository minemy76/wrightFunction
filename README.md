# wrightFunction

Данная программа является реализацией алгоритма расчета значений функции Райтта

Основная информация была взята из статьи(формула 1): https://www.researchgate.net/publication/331226733_The_Wright_function_and_its_applications

Подсчет гамма функии происходит при использовании известных апроксимаций на отрезке [0, 1], подробнее в файле constants.h

//

This program implements an algorithm for calculating the Wright function.

The main information was taken from the article (formula 1): https://www.researchgate.net/publication/331226733_The_Wright_function_and_its_applications

The gamma function is calculated using known approximations on the interval [0, 1]; more details are in the constants.h file.
